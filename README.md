# 🌟 今日の光スコア診断

今のあなたの心の状態を可視化し、心に灯をともすメッセージを届けるメンタルセルフケア・アプリです。

## 📋 アプリの概要
「ストレス」「集中」「静けさ」の3つの指標をスライダーで入力することで、独自の「光スコア」を算出します。スコアに応じて、今のあなたに必要な「おまじない」メッセージを表示します。
こちら↓
https://sorakinu.github.io/my-light-score-app

## ✨ 主な機能
- **直感的な入力**: スライダーを動かすだけで今の状態を評価。
- **光スコア表示**: 合計値を5.0満点に換算して表示。
- **パーソナライズ・メッセージ**: 
  - スコアが高い（4.0以上）ときは、祝福の**風船（バルーン）**が飛びます！
  - スコアが低いときは、心を癒やす4つの「おまじない」からランダムにアドバイスを表示。
- **シンプルで使いやすい**: 誰でもすぐに診断できる1ページ構成。

## 🛠 使用技術
- **Python 3.x**
- **Streamlit**: Webアプリケーションフレームワーク
- **Random**: アドバイスのランダム表示に使用

## 🚀 使い方
1. スライダーを今の気分に合わせて動かします。
2. 「診断する」ボタンをクリック。
3. 今日のスコアと、あなたへのメッセージを確認してください。

## 🔒 プライバシーポリシー（個人情報の取り扱い）

**1. 個人情報の収集について**
本アプリでは、氏名やメールアドレスなどの個人情報を収集する機能はありません。

**2. データの利用目的**
スライダーで入力された数値は、その場でのスコア計算にのみ使用されます。

**3. 第三者への提供**
収集している情報自体がないため、第三者にデータを提供することはありません。
